Projeto desenvolvido com HTML, CSS e JavaScript puro, que consome a API pública de Rick and Morty e exibe cards dinâmicos dos personagens da série.

Tecnologias utilizadas:

HTML5 → estrutura da página

CSS3 → estilização e layout responsivo

JavaScript (Fetch API) → consumo de dados e criação dinâmica dos elementos

Funcionalidades:

Consumo de dados reais da Rick and Morty API

Criação automática de cards com nome, imagem, status e espécie dos personagens

Interface responsiva e visual moderna
